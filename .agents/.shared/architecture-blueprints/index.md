# Bản vẽ Kiến trúc Tổng thể (Architecture Blueprints)

> File `index.md` này đóng vai trò là cửa ngõ để AI nắm bắt các tiêu chuẩn/kiến trúc (DNA) liên quan đến `architecture-blueprints`.

## Mục đích
Lưu trữ sơ đồ C4 Model, Architecture Decision Records (ADRs) và triết lý kiến trúc cốt lõi. MK12 (System Architect) sẽ quản lý và các Agent khác phải tuân thủ để tránh làm vỡ cấu trúc dự án.

## Hướng dẫn sử dụng
- **Không tự ý suy đoán:** Mọi triển khai liên quan đến domain này phải tuân theo các tài liệu nằm trong thư mục này.
- Nếu quy định chưa có, hãy yêu cầu User xác nhận và bổ sung tài liệu mới vào đây.
