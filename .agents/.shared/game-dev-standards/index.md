# Tiêu chuẩn Game Development (Game Dev Standards)

> File `index.md` này đóng vai trò là cửa ngõ để AI nắm bắt các tiêu chuẩn/kiến trúc (DNA) liên quan đến `game-dev-standards`.

## Mục đích
Game Design Documents (GDD), quy chuẩn tối ưu Assets, kiến trúc ECS/OOP. Dành riêng cho tác vụ của MK17.

## Hướng dẫn sử dụng
- **Không tự ý suy đoán:** Mọi triển khai liên quan đến domain này phải tuân theo các tài liệu nằm trong thư mục này.
- Nếu quy định chưa có, hãy yêu cầu User xác nhận và bổ sung tài liệu mới vào đây.
