# Tiêu chuẩn Mobile App (Mobile Standards)

> File `index.md` này đóng vai trò là cửa ngõ để AI nắm bắt các tiêu chuẩn/kiến trúc (DNA) liên quan đến `mobile-standards`.

## Mục đích
Các quy tắc thiết kế, build và deploy dành riêng cho ứng dụng di động (React Native/Expo/Flutter). Tiêu chuẩn bắt buộc cho MK09.

## Hướng dẫn sử dụng
- **Không tự ý suy đoán:** Mọi triển khai liên quan đến domain này phải tuân theo các tài liệu nằm trong thư mục này.
- Nếu quy định chưa có, hãy yêu cầu User xác nhận và bổ sung tài liệu mới vào đây.
