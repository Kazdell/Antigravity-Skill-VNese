# Tiêu chuẩn Tích hợp & MCP (Integration & MCP Protocols)

> File `index.md` này đóng vai trò là cửa ngõ để AI nắm bắt các tiêu chuẩn/kiến trúc (DNA) liên quan đến `integration-mcp-protocols`.

## Mục đích
Các nguyên tắc bảo mật API Key, cấu trúc MCP server, và chuẩn kết nối 3rd-party services. Cẩm nang của MK11.

## Hướng dẫn sử dụng
- **Không tự ý suy đoán:** Mọi triển khai liên quan đến domain này phải tuân theo các tài liệu nằm trong thư mục này.
- Nếu quy định chưa có, hãy yêu cầu User xác nhận và bổ sung tài liệu mới vào đây.
